Компоненты формы¶

Компонент — это единичный структурный элемент с определенным поведением. Поведение зависит от задачи, для которой он создан. При необходимости компоненты могут дополняться.

Все стандартные компоненты формы:

  • Список c единичным выбором
  • Список с множественным выбором
  • Единичный чекбокс
  • Пункт чеклиста
  • Список с выбором из множества значений
  • Дата
  • Дата и временя
  • Email
  • Числовые значения
  • Ниспадающий список
  • Многострочный текст
  • Краткий текст
