Текстовые элементы¶

Простое текстовое поле¶

text — текстовое поле.

"name": {
        "title": "Название",
        "description": "Название объекта",
        "type": "text",
        "minLength": 3
}

Текстовое поле в Конструкторе

Поле для ввода email¶

email — поле почты. Используется с валидацией формата email.

"email": {
    "title": "E-mail ответственного лица",
    "type": "email",
}
