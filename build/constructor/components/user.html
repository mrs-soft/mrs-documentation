Выбор пользователя «СтройКонтроля»¶

С помощью компонента можно выбриать пользователей из базы данных «СтройКонтроля».

Как задать компонент формы?¶

"selectUser": {
                "type": "user",
                "filters": {
                    "name": "Гаранин Тимофей",
                    "organizationId": "5fe33b42d017d966a5c721df"
                }
}
