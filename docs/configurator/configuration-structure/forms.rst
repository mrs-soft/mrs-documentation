Формы
=====

Форма позволяет создавать модели данных и бизнес-объекты для реализации целевых бизнес-процессов. 

Форма представляет собой описание модели бизнес-объекта в формате JSON. Форма имеет имя, настройки, схему данных. 

Схема данных состоит из именованных полей данных (атрибутов) определенного типа. Типом поля данных может быть один из компонентов формы.
Компонент задает как тип данных и структуру, так и визуальное отображение поля формы в пользовательском интерфейсе.

Форма также позволяет задавать правила целостности и поведения бизнес-объекта с помощью бизнес-правил — хуков.
