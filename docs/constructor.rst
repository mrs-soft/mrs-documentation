Конфигурация
============

Конфигурация реализует конкретное прикладное решение, работающее на МРС Платформе 1.0.
Она является расширением МРС Платформы 1.0 и задает поведение объектов Базового ПО для решения прикладной задачи.

С точки зрения разработки Конфигурация представляет собой репозиторий кода,
содержащий описание модели бизнес-объектов, бизнес-правила, запросы, сценарии интеграции и тд.
Разработка производится на языке JavaScript с использованием JSON-объектов для описания моделей и конфигурации, SQL-запросов.

Таким образом Конфигурация представляет собой программу для ЭВМ и может быть зарегистрирована
в качестве результатов интеллектуальной деятельности на основании ст 1232 ГК РФ.
